# GitHub Personal Access Token 복사 방법

## 📋 단계별 가이드

### 1단계: GitHub 웹사이트 접속
1. 브라우저에서 https://github.com 접속
2. GitHub 계정으로 로그인

### 2단계: Settings로 이동
1. 우측 상단 프로필 아이콘 클릭 (원형 아이콘)
2. 드롭다운 메뉴에서 **"Settings"** 클릭

### 3단계: Developer settings로 이동
1. 왼쪽 메뉴 맨 아래로 스크롤
2. **"Developer settings"** 클릭

### 4단계: Personal access tokens로 이동
1. 왼쪽 메뉴에서 **"Personal access tokens"** 클릭
2. **"Tokens (classic)"** 클릭

### 5단계: 토큰 생성 (이미 했다면 6단계로)
1. **"Generate new token"** 클릭
2. **"Generate new token (classic)"** 클릭
3. 설정:
   - **Note**: `Hana Project` 입력
   - **Expiration**: `1 year` 선택
   - **Scopes**: `repo` 체크박스 선택
4. 맨 아래 **"Generate token"** 버튼 클릭

### 6단계: 토큰 복사 (중요!)
1. 생성된 토큰이 화면에 표시됩니다
2. 토큰은 `ghp_`로 시작하는 긴 문자열입니다
3. **토큰 옆에 있는 복사 아이콘 클릭** (클립보드 모양)
   - 또는 토큰 전체를 드래그해서 선택
   - Ctrl + C (또는 우클릭 → 복사)

### 7단계: 안전하게 보관
- ⚠️ **이 페이지를 벗어나면 토큰을 다시 볼 수 없습니다!**
- 토큰을 메모장이나 안전한 곳에 저장해두세요
- 또는 바로 PowerShell에 붙여넣기

## 🔍 토큰이 보이지 않는다면?

### 이미 생성한 토큰 찾기
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. 생성한 토큰 목록 확인
3. 토큰 옆에 **"..."** (점 3개) 클릭
4. **"Regenerate"** 클릭 (새 토큰 생성)
5. 새 토큰 복사

### 토큰을 잃어버렸다면
- 토큰은 한 번만 보여줍니다
- 잃어버렸다면 새로 생성해야 합니다
- 기존 토큰은 취소하고 새로 만드세요

## 📝 토큰 모양 예시

토큰은 다음과 같이 생겼습니다:
```
ghp_1234567890abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
```

- `ghp_`로 시작
- 약 40자 정도의 긴 문자열
- 영문자와 숫자로 구성

## ✅ 복사 확인 방법

### 방법 1: 메모장에 붙여넣기
1. 메모장 열기
2. Ctrl + V (붙여넣기)
3. 토큰이 제대로 붙여넣어졌는지 확인

### 방법 2: PowerShell에 붙여넣기
1. PowerShell에서 Password 입력란 클릭
2. Ctrl + V (붙여넣기)
3. 토큰이 입력되었는지 확인 (보안상 `*`로 표시됨)
