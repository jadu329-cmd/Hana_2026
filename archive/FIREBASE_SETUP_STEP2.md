# Firebase 설정 2단계: Firestore Database 활성화

## 1. Firestore Database로 이동
1. Firebase Console에서 **"team-hana"** 프로젝트가 열려있는지 확인
2. 왼쪽 메뉴에서 **"Firestore Database"** 클릭

## 2. 데이터베이스 생성
1. "데이터베이스 만들기" 버튼 클릭

## 3. 보안 규칙 선택
두 가지 옵션이 있습니다:

### 옵션 A: 테스트 모드 (개발용, 빠른 시작)
- **"테스트 모드에서 시작"** 선택
- 30일 후 자동으로 거부되지만, 개발 중에는 편리합니다
- **권장**: 처음 시작할 때는 이 옵션 선택

### 옵션 B: 프로덕션 모드 (안전)
- **"프로덕션 모드에서 시작"** 선택
- 보안 규칙을 직접 설정해야 합니다
- 나중에 `firestore.rules` 파일로 규칙을 업데이트할 수 있습니다

## 4. 위치 선택
- **권장**: `asia-northeast3` (서울) 선택
- 또는 가장 가까운 지역 선택
- "사용" 클릭

## 5. 활성화 완료
- 데이터베이스 생성이 완료되면 Firestore Database 페이지가 표시됩니다
- 아직 컬렉션이 없어도 괜찮습니다 (앱에서 자동으로 생성됩니다)
