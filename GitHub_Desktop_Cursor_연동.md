# GitHub Desktop + Cursor 연동 가이드

## ✅ 네, 가능합니다!

GitHub Desktop과 Cursor를 함께 사용할 수 있습니다.
GitHub Desktop은 파일을 관리하고, Cursor는 코드를 편집합니다.

## 🔄 워크플로우

### 1단계: GitHub Desktop에서 프로젝트 열기
1. GitHub Desktop 실행
2. 프로젝트 선택: `Hana_2026`
3. **"Repository"** → **"Open in Cursor"** 클릭
   - 또는 **"Repository"** → **"Open in Explorer"** 클릭 후 Cursor로 폴더 열기

### 2단계: Cursor에서 파일 수정
1. Cursor에서 `Hana_council.html` 열기
2. 코드 수정
3. 파일 저장 (Ctrl + S)

### 3단계: GitHub Desktop에서 변경사항 확인
1. GitHub Desktop으로 돌아가기
2. 왼쪽에서 변경된 파일 자동 감지됨
3. 변경 내용 확인

### 4단계: 커밋 및 푸시
1. 하단에 커밋 메시지 입력 (예: "심방리스트 수정")
2. **"Commit to main"** 클릭
3. **"Push origin"** 클릭
4. 완료!

## 🎯 두 가지 방법

### 방법 1: Cursor에서 직접 열기
1. Cursor 실행
2. **"File"** → **"Open Folder"**
3. 프로젝트 폴더 선택: `C:\Users\park.ji.hun\Desktop\CODE\HANA`
4. 파일 수정
5. GitHub Desktop에서 변경사항 확인

### 방법 2: GitHub Desktop에서 Cursor 열기
1. GitHub Desktop에서 프로젝트 선택
2. **"Repository"** → **"Open in Cursor"** 클릭
   - (Cursor가 설치되어 있으면 이 옵션이 보입니다)
3. Cursor에서 파일 수정
4. GitHub Desktop에서 변경사항 확인

## 🔗 GitHub Desktop에서 Cursor 열기 설정

### Cursor를 기본 에디터로 설정
1. GitHub Desktop 실행
2. **"File"** → **"Options"** (또는 **"Preferences"**)
3. **"Integrations"** 탭 선택
4. **"External Editor"** 선택
5. **"Cursor"** 선택
6. 저장

이제 **"Repository"** → **"Open in Cursor"** 옵션이 나타납니다!

   - 파일 열기
   - 코드 편집

   - 커밋 및 푸시

## ⚠️ 주의사항

### 파일 저장 필수
- Cursor에서 수정 후 **반드시 저장** (Ctrl + S)
- 저장하지 않으면 GitHub Desktop에서 변경사항이 감지되지 않습니다

### GitHub Desktop 새로고침
- 변경사항이 보이지 않으면
- **"Repository"** → **"Refresh"** 클릭
- 또는 자동으로 감지됩니다
