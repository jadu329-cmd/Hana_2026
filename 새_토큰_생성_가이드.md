# 새 Personal Access Token 생성 가이드

## ⚠️ 토큰을 복사하지 않았다면?

GitHub에서 토큰은 생성 후 **한 번만** 보여줍니다.
복사하지 않았다면 **새로 생성**해야 합니다.

## 📋 새 토큰 생성 단계

### 1단계: GitHub 웹사이트 접속
1. 브라우저에서 https://github.com 접속
2. GitHub 계정으로 로그인

### 2단계: Settings로 이동
1. 우측 상단 프로필 아이콘 클릭 (원형 아이콘)
2. 드롭다운 메뉴에서 **"Settings"** 클릭

### 3단계: Developer settings로 이동
1. 왼쪽 메뉴 맨 아래로 스크롤
2. **"Developer settings"** 클릭

### 4단계: Personal access tokens로 이동
1. 왼쪽 메뉴에서 **"Personal access tokens"** 클릭
2. **"Tokens (classic)"** 클릭

### 5단계: 새 토큰 생성
1. **"Generate new token"** 클릭
2. **"Generate new token (classic)"** 클릭

### 6단계: 토큰 설정
1. **Note**: `Hana Project` 입력 (용도 표시)
2. **Expiration**: `1 year` 선택 (또는 원하는 기간)
3. **Scopes**: 
   - ✅ **`repo`** 체크박스 선택 (가장 중요!)
   - 이렇게 하면 모든 저장소 권한이 부여됩니다

### 7단계: 토큰 생성
1. 맨 아래로 스크롤
2. **"Generate token"** 버튼 클릭

### 8단계: 토큰 복사 (중요!)
1. 생성된 토큰이 화면에 표시됩니다
2. 토큰은 `ghp_`로 시작하는 긴 문자열입니다
3. **토큰 옆에 있는 복사 아이콘 클릭** (클립보드 모양)
   - 또는 토큰 전체를 드래그해서 선택
   - **Ctrl + C** (복사)

### 9단계: 안전하게 보관
- ⚠️ **이 페이지를 벗어나면 토큰을 다시 볼 수 없습니다!**
- 토큰을 메모장에 붙여넣어 저장하거나
- 바로 PowerShell에 사용하세요

## 🔍 기존 토큰이 있다면?

### 기존 토큰 확인
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. 생성한 토큰 목록 확인
3. 토큰 옆에 **"..."** (점 3개) 클릭
4. **"Regenerate"** 클릭 (새 토큰 생성)
5. 새 토큰 복사

## 📝 토큰 모양 예시

토큰은 다음과 같이 생겼습니다:
```
ghp_1234567890abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
```

- `ghp_`로 시작
- 약 40자 정도의 긴 문자열
- 영문자와 숫자로 구성

## ✅ 복사 확인 방법

### 방법 1: 메모장에 붙여넣기
1. 메모장 열기
2. Ctrl + V (붙여넣기)
3. 토큰이 제대로 붙여넣어졌는지 확인

### 방법 2: PowerShell에 바로 사용
1. PowerShell에서 Password 입력란 클릭
2. Ctrl + V (붙여넣기)
3. 토큰이 입력되었는지 확인 (보안상 `*`로 표시됨)
