# 배포 확인 및 실행 가이드

## 배포가 안 되었다면

### cmd 창에서 직접 실행

1. **Windows 키 + R**
2. **`cmd` 입력 후 Enter**
3. **다음 명령어 순서대로 실행:**

```cmd
cd C:\Users\park.ji.hun\Desktop\CODE\HANA
```

```cmd
firebase use team-hana
```

```cmd
firebase deploy --only hosting
```

### 배포 성공 확인

배포가 성공하면 다음과 같은 메시지가 표시됩니다:

```
✔  Deploy complete!

Hosting URL: https://team-hana.web.app
```

### 배포 실패 시 확인사항

1. **Firebase 로그인 확인**
   ```cmd
   firebase login:list
   ```
   로그인된 계정이 표시되어야 합니다.

2. **프로젝트 확인**
   ```cmd
   firebase projects:list
   ```
   `team-hana` 프로젝트가 보여야 합니다.

3. **오류 메시지 확인**
   - cmd 창에 표시된 오류 메시지를 확인하세요
   - 오류 메시지를 알려주시면 해결 방법을 안내하겠습니다

cd C:\Users\park.ji.hun\Desktop\CODE\HANA && firebase use team-hana && firebase deploy --only hosting
