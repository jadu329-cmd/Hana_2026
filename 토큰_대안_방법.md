# GitHub 토큰 없이 사용하는 방법

## 🤔 토큰을 안 쓰면 안 되나요?

### 답변: HTTPS로 푸시하려면 토큰이 필수입니다
- GitHub가 2021년부터 비밀번호 인증을 완전히 차단했습니다
- HTTPS로 푸시하려면 토큰이 필요합니다
- 하지만 **대안이 있습니다!**

## ✅ 토큰 없이 사용하는 방법

### 방법 1: SSH 키 사용 (권장!)

**장점:**
- ✅ 토큰 불필요
- ✅ 비밀번호 입력 불필요
- ✅ 한 번 설정하면 계속 사용 가능
- ✅ 더 안전함

**단점:**
- ⚠️ 초기 설정이 조금 복잡함

**설정 방법:**
1. SSH 키 생성
2. GitHub에 SSH 키 등록
3. 저장소 URL을 SSH로 변경

**장점:**

- ✅ 토큰 자동 관리

- ✅ 충돌 해결 쉬움

**단점:**
- ⚠️ 별도 프로그램 설치 필요

**설정 방법:**
1. GitHub Desktop 다운로드
2. GitHub 계정으로 로그인
3. 자동으로 인증됨

### 방법 3: GitHub CLI 사용

**장점:**
- ✅ 명령어로 쉽게 사용
- ✅ 토큰 자동 관리
- ✅ GitHub 기능을 명령어로 사용 가능

**단점:**
- ⚠️ 별도 프로그램 설치 필요

## 🔐 SSH 키 설정 방법 (상세)

### 1단계: SSH 키 생성

PowerShell에서:
```powershell
ssh-keygen -t ed25519 -C "your_email@example.com"
```

- Enter 누르기 (기본 위치 사용)
- Enter 누르기 (비밀번호 없음)
- Enter 누르기 (비밀번호 확인)

### 2단계: SSH 키 복사

```powershell
cat ~/.ssh/id_ed25519.pub
```

출력된 키 전체를 복사

### 3단계: GitHub에 SSH 키 등록

1. GitHub → Settings → SSH and GPG keys
2. "New SSH key" 클릭
3. Title: `My Computer` 입력
4. Key: 복사한 키 붙여넣기
5. "Add SSH key" 클릭

### 4단계: 저장소 URL 변경

```powershell
cd C:\Users\park.ji.hun\Desktop\CODE\HANA
git remote set-url origin git@github.com:jadu329-cmd/Hana_2026.git
```

### 5단계: 테스트

```powershell
git push -u origin main
```

비밀번호나 토큰 없이 작동합니다!

## 🖥️ GitHub Desktop 사용 방법

### 1단계: 다운로드
- https://desktop.github.com/ 접속
- 다운로드 및 설치

### 2단계: 로그인
- GitHub 계정으로 로그인
- 자동으로 인증됨

### 3단계: 저장소 추가
- "Add" → "Add Existing Repository"
- 프로젝트 폴더 선택

### 4단계: 사용
- 파일 수정 후
- 왼쪽에서 변경사항 확인
- "Commit to main" 클릭
- "Push origin" 클릭

끝! 토큰 입력 불필요!
